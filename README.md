# One's Own Room

A virtual safe room. 

Built for Meta Quest 3, playable in any WebXR browser.

## Stack

Three.js 0.178 + [Spark](https://sparkjs.dev) (Gaussian splatting, dyno GPU shaders) · WebXR · Marble (pre-generated splat worlds) · Claude (script generation + language detection) · ElevenLabs (TTS + STT).
